# GENI-ELOMOS-PRO

Application IA de prédiction sportive :
- Football
- Basketball
- Tennis
- Hockey sur glace
- Cricket

Fonctionnalités :
- Victoire probable
- Score exact
- Mi-temps / Fin
- BTTS (Les deux équipes marquent)
- Over/Under 1.5
- Over/Under 2.5
- Over/Under 3.5
- Over/Under 4.5
- Corners
- Cartons
- Alerte penalty
- Alerte carton rouge

Version Android avec API IA intégrée.
